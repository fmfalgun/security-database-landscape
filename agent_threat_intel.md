### 1. VirusTotal / Google Threat Intelligence (GTI)
- **Organization:** Google (Alphabet)
- **Product:** Google Threat Intelligence (GTI) — formerly VirusTotal, rebranded/expanded 2024
- **Website:** https://www.virustotal.com / https://cloud.google.com/security/products/threat-intelligence / https://gtidocs.virustotal.com
- **What data:** File hashes (MD5/SHA1/SHA256), URLs, domains, IPs, malware behavioral analysis (sandbox), YARA rules, threat actor profiles, campaign tracking, IOC collections, crowdsourced antivirus detections (70+ AV engines)
- **Data sources:** Crowdsourced submissions (3M+ files/URLs per day), 70+ AV engine results, sandbox dynamic analysis, Mandiant intel, Google's own threat intel (Safe Browsing, Gmail, etc.), community YARA rules, VirusTotal feeds
- **Scale:** 3M+ suspicious files/URLs submitted daily; hundreds of millions of file/URL/IP/domain records
- **Freshness:** Near real-time (live submission scanning)
- **Access model:** Free tier (web UI, limited API); paid API tiers; enterprise GTI platform (subscription); integrates with Google Cloud SecOps
- **Who uses it:** SOC analysts globally, threat researchers, IR teams, AV vendors, enterprise security teams, academic researchers
- **STIX/TAXII/MISP:** Yes — API supports STIX/TAXII export; integrates with Elastic, Google SecOps, SOAR platforms
- **Notes:** GTI merges VirusTotal's crowdsourced file/URL intel with Mandiant's expert APT intelligence and Google's global telemetry. Livehunt (real-time YARA matching) and Retrohunt (historical search) are premium features. IOC Stream provides consolidated feed for SIEM/SOAR ingestion.
